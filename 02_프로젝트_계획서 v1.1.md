## 8. 기술 스택 요약

| 계층 | 선정 | 비고 |
| --- | --- | --- |
| LLM | **Gemini (Primary)** + **Claude Haiku 4.5 (Secondary)** | A/B 비교·태스크별 라우팅 |
| RAG 오케스트레이션 | **LangGraph** | State 기반 상태 머신, 조건부 분기, 멀티턴 Checkpointer |
| RAG 컴포넌트 | **LangChain** | Self-Querying Retriever, Prompt, LCEL Runnable |
| 임베딩 모델 | **Gemma4-2B** | 리뷰 태그 생성 및 벡터 임베딩 (로컬·무료) |
| 백엔드 | **FastAPI** *(추천 확정)* | LangChain·Supabase·비동기 최적 |
| 프론트 | **Streamlit** (MVP) → **Next.js** (Stretch) | |
| DB | **Supabase (PostgreSQL + pgvector, HNSW)** | Semantic + Keyword + Cache 단일화 |
| 외부 API | TMDB, **SerpAPI** | 영화 정보·실시간 그라운딩 |
| 데이터 수집 | **Playwright 단독** | Watcha 크롤링 (동적 렌더링·무한 스크롤 대응) |
| 배포 | Vercel (프론트) · Render (FastAPI) · Supabase Cloud | 무료 티어 |
| 협업 | GitHub · Notion · Discord | — |
